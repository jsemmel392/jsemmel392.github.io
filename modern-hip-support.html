Modern HIP — Support

Modern HIP is a steganography utility for Windows that lets you hide any file inside a PNG image using AES-256 encryption. If you need help or have a question, you're in the right place.

Frequently Asked Questions

How do I hide a file?

Switch the app to Hide Mode (Red). Click the browse buttons to select your carrier PNG and the file you want to hide. Enter a strong Security Key for AES-256-GCM encryption, then click "Hide my file." A new image will be generated containing your encrypted data.

How do I retrieve a hidden file?

Switch the app to Retrieve Mode (Green). Select the PNG image that contains your hidden data and enter the original Security Key. Click "Retrieve my file" to decrypt and save your hidden file back to your machine.

Why are only PNG files supported?

PNG is a requirement for secure steganography. Unlike JPEGs, which use "lossy" compression that destroys hidden data bits, PNG is a lossless format. This ensures that your encrypted payload remains bit-perfect and recoverable every time.

How large a file can I hide?

The maximum hidden file size depends on the dimensions of your carrier PNG. As a general guide, a 1920x1080 PNG can hold approximately 750KB of hidden data. Larger images can accommodate larger payloads.

Is my data safe?

Modern HIP encrypts your hidden file using AES-256 before embedding it in the image. The app has no internet connectivity and does not transmit any data. All operations are performed entirely on your local machine.

Does Modern HIP work on Windows 10?

Yes. Modern HIP is compatible with Windows 10 and Windows 11.
